v3UsersUserIdDevicesDeviceIdResourcesResourceNamePost

Send data to a device resource. This means that the resource defined in the device will be called with the information provided in the body. The body can be of type `application/json` if you want to send JSON data, or can be of type `application/octect-stream` if you want to send binary data to the device.
