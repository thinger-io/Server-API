This is the API

v1UsersUserIdDevicesDeviceIdStatsGet

Get information about the device, like its connection state and timestamp, its current IP address, and transmitted data. This method can be used with Server Sent Events (SSE) to get real-time updates about device stats, like tx and rx bytes. If you cannot add the Authorization header to the SSE client, you can add the access token as a url parameter, like ?authorization=eyJhbGci....

URL

{protocol}://api.thinger.io//v1/users/{user_id}/devices/{device_id}/stats

HTTP Method

GET

Response Type

inline_response_200

Parameters

  • Path: user_id String

  • Path: device_id String
